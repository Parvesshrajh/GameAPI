🎮 Pokémon Quiz Game

A fun and interactive web-based quiz game where users guess Pokémon names based on their images. Built using HTML, CSS, and JavaScript, this project uses a public API to fetch real Pokémon data.

🚀 Features
🎲 Random Pokémon image each round
🔘 Multiple choice answers (4 options)
🧠 Score tracking system
🔟 10-question game limit
📊 Final performance evaluation
🔄 “Play Again” option after game ends
⚡ Uses Fetch API with async/await

🕹️ How to Play
A random Pokémon image will appear
Choose the correct name from 4 options
Score increases for each correct answer
Game ends after 10 questions
Final result will be shown:
✅ Score ≥ 5 → Good Pokémon knowledge
🙂 Score < 5 → Average knowledge
Click Play Again to restart
